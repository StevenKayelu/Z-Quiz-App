
# Z-Quiz App

This is my Android quiz app project called **Z-Quiz**. It was built using Kotlin in Android Studio as part of my Mobile Programming course.

## About the App

Z-Quiz is a simple quiz application where users can:
- Choose a quiz category
- Select difficulty level
- Choose number of questions
- Set time for the quiz
- Start the quiz and answer multiple choice questions
- View their final score at the end

## Features

- Simple and clean UI
- Custom quiz settings before starting
- Multiple choice questions (A, B, C, D)
- Score display at the end of the quiz

## Technologies Used

- Kotlin
- Android Studio
- XML for layouts
- Git & GitHub for version control

## How to Run the Project

1. Clone the repository
git clone https://github.com/your-username/Z-Quiz-App.git
2. Open the project in Android Studio
3. Let Gradle finish syncing
4. Run the app on an emulator or phone
   
# Project Structure
MainActivity → where you select quiz settings
QuizActivity → where questions are shown
ResultActivity → shows final score
Questions → stored locally in JSON / model class
